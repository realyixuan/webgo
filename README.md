# webgo

<img style="float: left;" src="https://img.shields.io/badge/pypi-0.1-green">

A micro web framework

You can imitate the `demo` directory to build your own.

**install and run it**

~~~bash
>>> pip install webgo
>>> python -m webgo demo
Serving demo__main__ ...
~~~

and access: http://localhost:8080
